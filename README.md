# VersusMode (GD 2.2)
A competitive multiplayer mod for **Geometry Dash** powered by the **Globed API**.

## ⚔️ How it Works
1. **The Button**: Replaces the inactive "Adventure" button in the main menu with a functional **Versus** button.
2. **Matchmaking**: Connects two players using Globed's networking.
3. **Voting**: Players have **15 seconds** to vote between **Classic** or **Platformer** modes.
4. **Random Selection**: If no vote is cast, the system uses the Globed Room ID as a seed to pick a random level for both players.
5. **Score to Win**: First player to reach **20 points** (based on level stars) wins the match!

## 🛠️ Installation
* Requires **Geode** and **Globed**.
* Download the `.geode` file from the Releases page and drop it into your GD mods folder.
